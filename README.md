# 청소년 연합 수련회 후원 페이지

GitHub Pages에 바로 올릴 수 있는 정적 후원 안내 페이지입니다.

## 구성

- `index.html`: 페이지 본문
- `styles.css`: 전체 디자인
- `assets/`: 수련회 사진

## GitHub Pages 사용 방법

1. 새 GitHub 저장소를 만듭니다.
2. 이 폴더의 파일을 저장소 루트에 올립니다.
3. 저장소의 `Settings > Pages`에서 배포 브랜치를 선택합니다.
4. 공개된 Pages 주소를 후원 안내 링크로 공유합니다.

계좌번호 복사 버튼은 HTTPS로 배포된 GitHub Pages에서 정상 동작합니다.
